### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（老师）、管理员。

**普通用户功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 班级管理
5. 教学资料管理
6. 课程管理
7. 学生管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 班级管理
4. 基础数据管理（资料类型管理、课程类别管理、院系管理）
5. 教学资料管理
6. 课程管理
7. 老师管理
8. 学生管理
